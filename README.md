## Challenge Adevinta

### To run test
 `mvn test`

### To compile
`mvn package` 
### To run app

`java -cp target/challenge-1.0-SNAPSHOT.jar com.adevinta.App <PATH/TO/FILES>
` 

Notes:
1. Maven 3 is used to build
2. Java 1.8 was used, language level 8
3. search for a duplicate "search:> duplicate duplicate" will return 100% match even if the file only has the word once
